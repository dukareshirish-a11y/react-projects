📌 Internship Task – Mini Project week 1

This project is a simple Calculator Web Application developed using React.js during my internship practice.
The application performs basic arithmetic operations such as addition, subtraction, multiplication, and division with a clean and responsive user interface.

The main objective of this project is to understand React components, state management, and event handling while building a functional calculator.

🚀 Features

➕ Addition operation
➖ Subtraction operation
✖️ Multiplication operation
➗ Division operation
🧮 Real-time calculation
🎨 Clean and simple UI
⚡ Fast and responsive performance
📱 Beginner-friendly project
